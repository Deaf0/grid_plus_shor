# Минимизация расстояния Хаусдорфа (только перенос)

Этот проект вычисляет оптимальный вектор переноса между двумя 2D-многоугольниками, минимизирующий расстояние Хаусдорфа.

## Пайплайн

1. **Инициализация**

   * Вычисляется область поиска `Q0` на основе ограничивающих прямоугольников (bounding box)

2. **Растеризация**

   * Многоугольники преобразуются в плотные облака точек

3. **Поиск по сетке (Grid Search)**

   * Расстояние Хаусдорфа вычисляется на грубой сетке по области `Q0`
   * Выбирается лучший кандидат

4. **Уточнение (метод Шора)**

   * Используется субградиентный спуск для уточнения вектора переноса

---

## Пример использования

Основной сценарий работы (см. `main.py`):

```python id="k3n92x"
# 1. Инициализация области поиска
Q0 = initQ0(A, B)

# 2. Растеризация многоугольников
A_rasterize = rasterize_polygon(A, 50)
B_rasterize = rasterize_polygon(B, 50)

# 3. Построение KD-деревьев
A_tree = KDTree(to_array(A_rasterize))
B_tree = KDTree(to_array(B_rasterize))

# 4. Грубый поиск оптимального сдвига
best_x, best_val, _ = find_optimal_translation_grid(
    A_rasterize,
    B_rasterize,
    A_tree,
    B_tree,
    Q0,
    20
)

# 5. Уточнение решения методом Шора
refine_x, refine_dist = shor_optimize(
    A_rasterize,
    B_rasterize,
    A_tree,
    B_tree,
    best_x
)
```

## Примечания

* Расстояние Хаусдорфа аппроксимируется с использованием растеризованных многоугольников
* Для ускорения поиска ближайших соседей используется KDTree
* Точность результата зависит от плотности растеризации
